Look at the two agents and compare their permissions:

  Research Agent:  Read-only web access
  Release Agent:   Workspace file access (elevated)

The Release Agent trusts data from the Research Agent (check
its config and source code). But the Research Agent reads
from the web/ directory — which contains external, untrusted
content.

Read the agent source files. Look at:
  1. The Release Agent's system prompt — what does it tell
     the AI about research context?
  2. The Release Agent's BASE_DIR — what directory can it
     access?
  3. The Research Agent's browse function — does it sanitise
     web content before returning it?

The trust chain is: Web → Research Agent → Release Agent.
If web content contains hidden instructions, where do they
end up?
