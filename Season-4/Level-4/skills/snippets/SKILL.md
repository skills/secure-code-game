# Code Snippets

Save and recall useful code snippets in your workspace. Use when you need to store or retrieve reusable code patterns.

## Usage

- `run snippets list` — List all saved snippets
- `run snippets save <name> <code>` — Save a snippet under a short name
- `run snippets get <name>` — Retrieve a saved snippet
