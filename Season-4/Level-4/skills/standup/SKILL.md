# Standup Helper

Generate a standup summary template from your recent activity. Use when you need a quick daily update structure personalised from memory.

## Usage

- `run standup` — Generate a standup template for the current user
- Use `remember name=<YourName>` first to personalise the output
- Fill in yesterday, today, and blockers before sharing your update
